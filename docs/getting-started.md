# Getting Started

## English

**Install:** `Settings → Plugins → Marketplace`, search **MyBatisCodeAssistant**, click Install and restart.

**Free vs Paid:** Navigation, completion, inspections and formatting are **free**. Code generation and the MyBatis Log analyzer require a subscription (free trial available).

**Quick start (2 minutes):**

1. Open the **Database** tool window and connect your data source.
2. Right-click a table → **Mybatis generator**.
3. Confirm the packages, tick the options you need, click **OK** — Entity / Mapper / Service / Controller / SQL XML are generated in one click.
4. In a Mapper interface, type a method name like `selectAllByNameAndAge`, press **Alt+Enter** → generate the SQL XML statement automatically.
5. Re-generating never overwrites your hand-written code — see the "How files are merged" note in the generation dialog.

## 简体中文

**安装：** `Settings → Plugins → Marketplace`，搜索 **MyBatisCodeAssistant**，安装后重启 IDE。

**免费与订阅：** 跳转、补全、检查、格式化等功能**免费**；代码生成与 MyBatis Log 解析需要订阅（支持免费试用）。

**两分钟上手：**

1. 打开右侧 **Database** 工具窗口，连接你的数据源。
2. 右键任意一张表 → **Mybatis generator**。
3. 确认包路径、勾选需要的选项，点击 **OK** —— Entity、Mapper、Service、Controller、SQL XML 一键生成。
4. 在 Mapper 接口中输入 `selectAllByNameAndAge` 这类方法名，按 **Alt+Enter** → 自动生成对应 SQL XML 语句。
5. 重复生成不会覆盖你的手写代码 —— 详见生成弹窗里的"文件是怎么合并的"说明。
