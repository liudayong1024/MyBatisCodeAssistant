# MyBatisCodeAssistant

**简体中文** · [使用说明](user-guide_zh.md) &nbsp;|&nbsp; **English** · [User Guide](user-guide_en.md)

---

一款 IntelliJ IDEA 的 MyBatis 全家桶代码助手:数据库表一键生成代码、方法名生成 SQL、智能合并、
智能补全、MyBatis 日志解析等。

An all-in-one MyBatis code helper for IntelliJ IDEA: one-click code generation from database tables,
method-name-to-SQL, smart merge, rich completion, MyBatis log analysis and more.

## 主要功能 / Main Features

数据库表一键生成 Entity/Mapper/Service/Controller/SQL XML · 方法名生成 SQL · 重复生成智能合并 ·
表/列与 `#{}` 参数智能补全 · OGNL 补全 · MyBatis Log 实时解析 · 动态 SQL 预览与交互式测试 ·
SQL 转 MyBatis · 自定义 Velocity 模板 · 数据库文档导出 等。

Code generation from database tables · method-name-to-SQL · smart merge on regeneration ·
table/column & `#{}` completion · OGNL completion · real-time MyBatis Log analyzer ·
dynamic SQL preview & interactive testing · SQL-to-MyBatis · custom Velocity templates ·
database documentation export, and more.
